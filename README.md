# react-native-ivs-player

Amazon Ivs player for react native

## Installation

```sh
npm install react-native-ivs-player
```

## Usage


```js
import { IvsPlayerView } from "react-native-ivs-player";

// ...

<IvsPlayerView color="tomato" />
```
